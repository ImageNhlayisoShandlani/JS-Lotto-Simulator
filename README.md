# Lotto Simulator
This is a simple lotto simulator built using HTML, CSS, and JavaScript. It allows
users to select their numbers and simulate a lotto draw to see if they win.
## Features
- Select numbers from a predefined range.
- Simulate a lotto draw with random numbers.
- Display the results of the draw and indicate if the user has won.
## Usage
1. Open the `index.html` file in a web browser.
